# Implementation of Grover's search

**Grover's search**
Before explain the implementation of Grover's search I'll explain what it is. Grover's search is an algorithm that uses quantum operations to find a specific item in an unsorted dataset much faster than a classical solution. Instead of searching each object one by one Grover's alorithm uses superposition to search all inputs simultaneously.
